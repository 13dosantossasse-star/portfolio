01 — PROFIL

À propos de moi

Étudiant en cybersécurité, je me spécialise dans le test d'intrusion Web. Mon approche est méthodique : reconnaissance, énumération, exploitation, puis rapport — comme dans un vrai engagement professionnel.

J'ai monté mon propre laboratoire virtuel sous VMware pour pratiquer sur des machines vulnérables (VulnHub), ce qui m'a permis de maîtriser les phases d'une attaque complète de bout en bout.

Mes outils de prédilection vont de Burp Suite pour la manipulation des requêtes HTTP, à SQLmap pour les injections, en passant par Metasploit pour l'exploitation post-compromission. Je code en Python pour automatiser mes scripts d'attaque.

Mon objectif : intégrer une équipe Red Team et contribuer à sécuriser des systèmes réels grâce à une vision offensive rigoureuse.
